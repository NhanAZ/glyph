# Glyph Tools

A lightweight, client-side toolkit for generating and exporting Minecraft glyphs and custom emoji assets. Upload or type hex, preview a 16×16 grid, and export strings or JSON packs—all without a backend.

## Live site
- https://nhanaz.github.io/glyph/

## Features
- Glyph generator with PNG upload, default template, and example assets.
- Smart converter for hex/emoji input with one-tap copy.
- Responsive grid: adaptive columns on mobile with an optional 16×16 toggle; desktop layout stays intact.
- Smart export menu: copy full grid, copy reference map, download JSON pack or metadata.
- Dark mode toggle and touch-friendly tooltips on mobile.

## Usage hints
- Type a hex code (default E0) or upload `glyph_XX.png` to populate the grid.
- Use the mobile grid toggle to force a full 16×16 view when space is tight.
- Click any cell to copy its character; tooltips show position and hex on hover/tap.
- "Smart Export" provides copy and JSON download options.
